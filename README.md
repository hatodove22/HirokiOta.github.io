# 博士課程学生ポートフォリオ V1

博士課程学生ポートフォリオ V1 の React + Vite 実装です。元デザインは次の Figma ドキュメントをご参照ください。
https://www.figma.com/design/q96wxyH2JhBsvZj0gPyVxD/%E5%8D%9A%E5%A3%AB%E8%AA%B2%E7%A8%8B%E5%AD%A6%E7%94%9F%E3%83%9D%E3%83%BC%E3%83%88%E3%83%95%E3%82%A9%E3%83%AA%E3%82%AA-V1

## 開発環境

```bash
npm install
npm run dev
```

Vite の開発サーバーが `http://localhost:3000/` で起動します。

## 本番ビルド

```bash
npm run build
```

生成物は `build/` ディレクトリに出力されます（Git には含めていません）。
