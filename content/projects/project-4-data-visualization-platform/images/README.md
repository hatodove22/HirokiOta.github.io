# プロジェクト画像

このフォルダには、データ可視化プラットフォームに関連する画像ファイルを格納します。

## ファイル構成

- `data-viz-hero.jpg` - メイン画像（カード表示用）
- `screenshots/` - プラットフォームのスクリーンショット
- `diagrams/` - システム構成図・フローチャート
- `results/` - 可視化結果のサンプル
- `ui-mockups/` - UIデザインのモックアップ

## 画像の命名規則

- ファイル名は英語で記述
- ハイフン（-）を使用して単語を区切る
- 例: `dashboard-overview-screenshot.png`
