# プロジェクト画像

このフォルダには、ロボティクス・コンピュータビジョンプロジェクトに関連する画像ファイルを格納します。

## ファイル構成

- `robot-vision.jpg` - メイン画像（カード表示用）
- `screenshots/` - システムのスクリーンショット
- `diagrams/` - システム構成図・フローチャート
- `results/` - 実験結果のグラフ・表
- `hardware/` - 使用したハードウェアの写真

## 画像の命名規則

- ファイル名は英語で記述
- ハイフン（-）を使用して単語を区切る
- 例: `object-detection-demo.png`
