# Issue: 論文コレクションのフィールドスキーマ

## 背景
論文コレクションのフィールド構成を確定し、後続のBibTeX連携やプロジェクト参照で利用できるデータを整える。

## スコープ
- 論文コレクションに必要なフィールド（タイトル、著者、出版年、DOIなど）をDecap設定に追加
- 必須項目バリデーションと、著者リストなどの複数項目入力の初期化処理を定義
- 既存BibTeXまたはMarkdownとのフィールドマッピングをドキュメント化

## 完了条件
- 論文コレクションの新規作成が可能で、必須項目未入力時には保存エラーになる
- プロジェクトからのリレーションに必要なフィールド（タイトル等）が必ず取得できる

- BibTeX同期の実処理はIssue 10で扱う
