# Issue: ニュース編集フォームとプレビュー実装

## 背景
ニュース投稿は最も頻度が高く、日付・タイトル・本文・サムネイルなど基本的な項目が必要。EditMode仕様に従って多言語対応や公開設定ドロワーを組み込み、プレビューと整合させる必要がある。

## スコープ
- Decap CMSのニュースコレクションに必要なフィールド定義を追加（タイトル、公開日、サマリー、本文、タグ、公開フラグ、多言語フィールド、サムネイル画像など）
- 保存時バリデーション（公開対象言語の必須チェック、画像alt必須など）を設定
- Reactプレビューコンポーネントをニュース用に実装し、実際のサイトデザインと差異が出ないよう調整
- ダミーデータを含むJSON出力テンプレートを作成し、プレビューが想定通り表示されるか確認

## 完了条件
- ニュースアイテムの作成・編集・下書き・公開がDecap CMS上で完了できる
- 多言語タブの入力必須チェックが期待通り動作する
- プレビューが現行サイトのニュース記事レイアウトと視覚的に一致している
