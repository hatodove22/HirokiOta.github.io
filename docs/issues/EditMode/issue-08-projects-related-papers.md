# Issue: プロジェクトと論文の関連参照モック連携

## 背景
プロジェクトは関連する論文を参照する仕様があり、CMS上でリンク関係を管理できるようにする必要がある。まずは論文コレクションの最低限のデータを参照できるモック連携を整える。

## スコープ
- プロジェクト編集フォームで関連論文を選択できるリレーションフィールドを追加（Decapのrelation widget）
- 論文側の必要最小限のフィールド（タイトル、発行年など）を仮定し、モックデータで参照できるようにする
- プロジェクトプレビューで関連論文一覧が表示されるようにし、データがない場合のフォールバックも用意

## 完了条件
- プロジェクト編集時に論文を検索・選択でき、保存後に再編集すると選択状態が維持されている
- プレビューで関連論文のタイトルが確認できる

## 依存/備考
- Issue 07でフィールド定義を完了していることが前提
- 論文コレクションの本格実装はIssue 09以降で行うため、必要なフィールドが増えた場合は後続で調整する
