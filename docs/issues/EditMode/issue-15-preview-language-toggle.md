# Issue: プレビューの言語切替UIと警告表示

## 背景
多言語バリデーションが整った後は、プレビューでも言語切替と警告表示を行い、入力者が不足している言語情報に気づけるようにする。

## スコープ
- プレビュー画面に言語切替UIを追加し、日本語/英語など切替できるようにする
- 多言語フィールドが欠落している場合に警告メッセージやプレースホルダーを表示
- ニュース・プロジェクト・論文のプレビューで共通UIが再利用できるよう、コンポーネント化

## 完了条件
- プレビューで言語切替を行うと対象言語の内容が表示される
- 欠落している言語がある場合は明確に警告が出て、公開前に気づける
- 共通コンポーネントが他コレクションでも流用可能な構造になっている

## 依存/備考
- Issue 14のバリデーション共通化が完了していることが前提
- 追加のアクセシビリティ対応は別Issueで検討する

## 方針更新（2025-09-29）
- プレビューの言語切替は Proto 内のタブ/トグルで提供。
- 欠落言語の警告は Proto のアラートUIで共通表示。
