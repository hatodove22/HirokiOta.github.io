# Issue: プロジェクトコレクションのフィールドスキーマ

## 背景
ニュースの縦切りを終えた後、次に扱うプロジェクトコレクションの基本フィールドとバリデーションを整え、データ入力のベースを固める。

## スコープ
- プロジェクトに必要な基本フィールド（タイトル、期間、説明、リポジトリURLなど）をDecap設定に追加
- 必須項目のバリデーションと、複数選択フィールド（カテゴリ等）がある場合の初期値/制限を定義
- 既存Markdownとの互換性確認および移行メモの追記

## 完了条件
- プロジェクトコレクションで新規エントリを作成し、必須項目が欠落していると保存できない
- 既存データをCMSに読み込んだ際に致命的な差分が発生しない

## 依存/備考
- プロジェクトと論文の関連付けは後続Issueで扱うため、本Issueではフィールド定義のみを対象とする
