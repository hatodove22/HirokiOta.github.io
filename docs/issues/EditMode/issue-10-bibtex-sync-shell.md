# Issue: BibTeX入出力フローの雛形

## 背景
論文コレクションはBibTeXとの同期が必要だが、いきなり完全な双方向連携を作るとリスクが高い。まずは最低限の入出力フローを整え、手動での同期を補助する。

## スコープ
- BibTeXファイルを解析してDecapコレクションにインポートするスクリプトの雛形を追加
- CMSからのエクスポート用に、論文データをBibTeX形式で出力する処理を用意（単一ファイルで可）
- スクリプトの使い方をDocsに記載し、手動実行で往復できることを確認

## 完了条件
- ローカルでスクリプトを実行するとBibTeX→CMS→BibTeXの往復ができ、データ欠損がない
- 自動化はしていないが、手順通りに実行すれば同期が可能
