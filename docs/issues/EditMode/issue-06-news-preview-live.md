# Issue: ニュース記事プレビューのLive更新

## 背景
ニュースコレクションのフィールドが揃ったため、Liveプレビューで公開時の見た目を確認できるようにする。プレビューの完成により、最初の縦切り（ニュースの作成→プレビュー→保存）が完結する。

## スコープ
- ニュース一覧/詳細コンポーネントを再利用したプレビューを実装し、入力内容が即時に反映されるようにする
- タイトル・本文・公開日・タグ等、主要フィールドがプレビューに表示されることを確認
- プレビュー内でエラーが発生した際のフォールバック表示（例: プレースホルダーテキスト）を追加

## 完了条件
- 入力値を変更すると数秒以内にプレビューへ反映される
- 保存されていないドラフトでもプレビューが機能する
- フィールド不足時にも編集が継続できるよう、プレビュー側で例外が発生しない
