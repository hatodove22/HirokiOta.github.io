# Issue: JSON生成をViteビルドに統合

## 背景
ローカルスクリプトでのJSON生成が整ったら、サイトのビルドプロセスと連携させ、CMSでの更新がデプロイに反映されるようにする。

## スコープ
- `npm run build` 実行時にJSON生成スクリプトが自動で実行されるよう、Viteまたはnpmスクリプトを調整
- 生成されたJSONの出力先を本番ビルドと整合させ、不要なファイルはクリーンアップ
- ビルド手順をDocsに更新し、必要な前提条件（環境変数など）があれば明記

## 完了条件
- `npm run build` を実行すると静的アセットとJSONが揃った状態で出力される
- ビルド失敗時にはJSON生成のログが確認でき、エラー原因が特定しやすい

## 依存/備考
- Issue 11でJSON生成スクリプトが完成していることが前提
- CIへの組み込みは次のIssueで対応する

## 方針更新（2025-09-29）
- `npm run build` 時に Proto 由来のJSONを確定・出力する前処理を組み込む。
- 生成手順/前提（環境変数・権限など）は Decap 依存を前提にしない。
