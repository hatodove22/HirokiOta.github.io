# Issue: プロジェクト編集フォームと関連論文リレーション

## 背景
プロジェクトは関連論文やタグ、公開ステータスなどニュースよりも多くのフィールドを持つ。EditMode仕様では`related_papers`の並び替えや多言語本文が求められているため、適切なUIとバリデーションを構築する必要がある。

## スコープ
- プロジェクトコレクションのフィールドをDecap CMS `list` + `relation` ウィジェットで設定し、ドラッグ&ドロップで順序を保持
- 基本情報（タイトル/ステータス/開始日/タグ/featured）のフォーム実装
- 多言語本文タブ、画像タブ、公開設定ドロワーの項目を仕様通りに整備
- プレビューで関連論文カードやメタ情報が正しく表示されるようReactコンポーネントを更新

## 完了条件
- プロジェクトの新規作成・編集時に関連論文を複数選択し並び替えられる
- 公開対象言語ごとの必須入力チェックと画像alt必須が動作
- プレビューが現行サイトのプロジェクト詳細画面と同等の内容を表示

## 依存/備考
- Issue #03でニュース向けの多言語/画像チェックが整備されていると実装がスムーズ
- 論文データのソースはIssue #05と連携するため、暫定的にモックデータで接続し、後続で置換
