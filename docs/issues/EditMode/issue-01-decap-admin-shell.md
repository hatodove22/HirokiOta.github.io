EditMode.mdの実装ではDecap CMSが編集UIの土台となる。まずはニュース編集だけでも動作確認できる最小限の管理画面を用意し、以降のフォーム実装を安全に進められる状態を整える。

## スコープ
- `/admin/index.html` と `config.yml` を追加し、ニュースコレクションのみを仮登録

- 必要なビルド/開発コマンドをREADMEまたは専用ドキュメントに追記

## 完了条件

- リポジトリの初回クローン後に追加のセットアップなしで上記の確認が可能
