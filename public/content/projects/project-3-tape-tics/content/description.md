# Tape-Tics — 「切って貼るだけ」の多点触覚テープ（AHs 2025）

## 背景
振動で合図や触感を伝えるウェアラブルは増えていますが、多点の振動子を「好きな場所・長さ」で素早く配置して試すのは、配線や筐体づくりが負担になりがちです。Tape-Ticsは、この試作の手間を大きく減らすために、フレキシブルな“テープ状”に振動子とLEDを並べ、必要な長さに切って曲面にも貼れる構造を採用しました。教育現場やXRプロトタイピングのように、短時間で多くの配置・パターンを試したい用途を主眼にしています。

## アプローチ
細長いフレキシブル基板（FPC）の帯に、小型の円盤型振動モータとフルカラーLEDを等間隔で実装し、ノード単位で個別に振動強度や点灯を制御できるようにしました。制御部は無線（BLE）で操作でき、GUIから直感的に強さや色、並びを設定できます。帯は任意の長さで切断して再結合でき、曲面や衣服にも追従します。テープLEDのような扱いやすさで、配線の煩雑さを避けつつ、貼るだけで多点の触覚表現を構成できる点が要です。

## 何がうれしいか
配線・筐体作りに時間を取られず、触覚デザインの「中身」に集中できます。曲面に沿って貼れるため、腕・手・背中など身体のさまざまな部位で配置の試行錯誤がしやすく、LEDとの連動で“見える触覚”の作り込みも容易です。ハンズオン形式の体験・教育では、参加者がGUIでパターンを構成し、理解が深まったとの評価が報告されています。

## 今後の展望
ノード数の拡張や、筋電・位置など他センサとの組み合わせにより、スポーツやリハビリ、XRインタラクション設計の実験速度をさらに高められます。既存の大規模ツールキットと併用し、教育・研究・アートなどの幅広い現場で活用されるプロタイピングツールとしての展開を目指しています。
