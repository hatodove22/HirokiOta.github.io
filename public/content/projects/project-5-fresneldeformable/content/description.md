# FresnelDeformable（フレネルデフォーマブル）

## 何を目指しているか

軽くてシンプルな仕組みで「柔らかさ」をもっと分かりやすく伝えることを目指しています。大掛かりな機構を使わず、視覚と触覚の手掛かりを組み合わせることで、手で触れたときの納得感を高めます。

## どんなアプローチか

指先が触れている小さな接触平面の角度を、押す力にあわせて少しだけ傾けます。目にはVR空間上で仮想物体がたわむ様子（擬似的な剛性表現）が見え、指先では接触面の傾き変化を感じます。視覚と触覚二つの手掛かりによって、あたかもやわらかい物体を持っているかのように錯覚させることができます。

## デバイスのイメージ

親指・人さし指・中指に対応した三つの小さな接触平面を、コンパクトなモータで角度制御します。物体を押す力を簡易的に推定し、その量に応じて指とデバイスの接触平面の傾きを切り替えます。装置自体は軽量で、既存のコントローラなどにも取り付けやすい構成です。

## これで何が嬉しいか

仮想物体の「見た目の変形」と「指先の傾き」という二つの小さな手掛かりだけで、複雑な機構を使わずに柔らかさの表現力を底上げできます。視覚と触覚の一貫性が高まり、VR空間への没入感や体験の向上が期待できます。
教育・デモ・展示など、機構の制約が厳しい場面でも扱いやすいのが強みです。

## 展望

より小型・省電力化し、既存のVR/触覚デバイスとの組み合わせを広げます。また、押し方や握り方などユーザの操作から、最適な傾き制御へ自動で切り替えるアルゴリズムの改良も進めます。
